# Assignment 2: Netflix User Analytics

## Overview
This assignment focuses on analyzing a Netflix User Analytics dataset using Python for data analysis and machine learning. The project demonstrates data preprocessing, exploratory data analysis, predictive modeling, and business insights.

## Objectives
- Load and explore the dataset.
- Perform exploratory data analysis (EDA).
- Prepare data for machine learning.
- Predict subscription renewal using classification models.
- Predict monthly spending using regression.
- Answer business-related questions based on the analysis.

## Topics Covered
### Data Analysis
- Dataset loading and inspection
- Descriptive statistics
- Missing value analysis
- User demographics and subscription analysis

### Data Preparation
- Categorical data encoding
- Feature and target selection
- Train-test split

### Machine Learning
- Decision Tree Classification
- K-Nearest Neighbors (KNN)
- Linear Regression

### Business Insights
- Subscription renewal factors
- Customer retention analysis
- Model performance comparison
- Practical business applications

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Outcome
This assignment provides hands-on experience with data preprocessing, visualization, machine learning, and business analytics using Python and Google Colab.
