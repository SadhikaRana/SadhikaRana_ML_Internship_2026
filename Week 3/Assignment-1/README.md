# Agriculture Yield Analysis

This project performs Exploratory Data Analysis (EDA) and basic Machine Learning on an agriculture dataset to identify factors affecting crop yield.

## Objectives
- Explore and understand the dataset.
- Analyze distributions and relationships between features.
- Compare crop and soil types based on average yield.
- Prepare data for machine learning.
- Build a Linear Regression model to predict crop yield.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Data inspection and cleaning
- Descriptive statistics
- Histograms and scatter plots
- Correlation analysis
- Crop and soil type comparisons
- Feature encoding
- Feature selection
- Train-test split
- Linear Regression model

## Key Insights
- Crop yield follows an approximately normal distribution.
- Rainfall shows a stronger relationship with yield than fertilizer usage.
- Rice has the highest average crop yield.
- Loamy soil has the highest average yield among soil types.
