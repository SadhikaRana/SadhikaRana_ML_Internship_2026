# Assignment 2

This assignment focuses on Python loops, conditional statements, user input, and basic problem-solving techniques.

## Programs

1. Sum of First 10 Natural Numbers
2. Factorial of a Number
3. Fibonacci Series
4. Largest Among Three Numbers
5. Student Result System
   - Student details input
   - Marks entry using loops
   - Percentage calculation
   - Grade determination using if-elif-else

## Concepts Used

- for loops
- if-elif-else statements
- User input using `input()`
- Arithmetic operations
- Variables and data types
- Basic algorithmic thinking

## Learning Outcomes

- Implement iterative solutions using loops
- Solve mathematical problems programmatically
- Work with conditional logic
- Build simple interactive console applications
- Organize Python programs with clear structure
