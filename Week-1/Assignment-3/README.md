# Assignment 3

This assignment focuses on Python functions, file handling, exception handling, and object-oriented programming.

## Programs Included

1. Print first 10 natural numbers.
2. Calculate the sum of first N natural numbers.
3. Reverse a number.
4. Count digits in a number.
5. Check whether a number is a palindrome.
6. Generate Fibonacci series.
7. Calculator using functions.
8. Store student details in a text file.
9. Read data from a text file.
10. Handle division by zero using exception handling.
11. Create a Student class with name and marks.

## Concepts Used

* Functions
* Loops
* Conditional statements
* User input
* File handling
* Exception handling
* Classes and objects

## Learning Outcome

This assignment demonstrates the implementation of basic Python programming concepts, modular programming using functions, file operations, exception handling, and object-oriented programming.
