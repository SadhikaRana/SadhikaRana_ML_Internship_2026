# Assignment 1

This assignment covers basic Python programming concepts including user input, arithmetic operations, variable manipulation, and simple problem-solving.

## Programs

1. Area of Rectangle
2. Simple Interest Calculator
3. Celsius to Fahrenheit Conversion
4. Average of Three Numbers
5. Square and Cube of a Number
6. Swap Two Numbers Without Third Variable
7. Student Report Program

## Learning Outcomes

- Taking user input using `input()`
- Using variables and data types
- Performing arithmetic calculations
- Writing clean and structured Python programs
- Understanding basic problem-solving techniques
